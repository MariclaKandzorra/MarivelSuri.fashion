This is a fully responsive & mobile friendly YouTube video background scaled with 16/9 aspect ratio.

To make this compatible with a CMS, you can pass data attributes into the video wrapper. This version only supports one video. If you'd like to see how this works with an array of videos, check out the version I originally forked.

It's important to note that this will not work on mobile devices, and there is no way to make it work. Why? Because mobile devices do not permit background video that is controlled by JS. I'm assuming this has something to do with bandwidth usage that cannot be controlled by the mobile device owner?
